Что это такое?

JavaScript — это язык фронтенда, то есть лицевой стороны сайта, которую видят пользователи. Чаще всего его используют в вебе для создания интерактивных страниц — с анимированными элементами, всплывающими сообщениями, блоками для отправки информации.
TypeScript — это доработанная версия JavaScript. Он помогает писать более чистый и безопасный код, так как предупреждает об ошибках ещё до запуска программы.

История создания

JavaScript был разработан в 1995 году Бренданом Айком в компании Netscape. Язык изначально создавался для добавления интерактивности на веб-страницы и быстро стал стандартом для веб-разработки. JavaScript является динамически типизированным языком, что означает, что типы переменных могут изменяться во время выполнения программы.

TypeScript был представлен Microsoft в 2012 году как надстройка над JavaScript. Разработчик языка, Андерс Хейлсберг, стремился решить проблемы, возникающие при разработке крупных приложений на JavaScript, такие как сложность отладки и поддержания кода. TypeScript добавляет статическую типизацию и поддержку объектно-ориентированного программирования, что делает код более предсказуемым и легким для понимания.

Как работает ?

JavaScript

JavaScript интерпретируется браузерами, что позволяет выполнять код непосредственно на стороне клиента. Он поддерживает динамическую типизацию, что делает его гибким, но может привести к ошибкам, которые обнаруживаются только во время выполнения.

TypeScript
TypeScript компилируется в JavaScript, что обеспечивает его выполнение в любом современном браузере или на серверной платформе, такой как Node.js. Статическая типизация позволяет находить ошибки на этапе компиляции, что облегчает разработку и поддержку кода.

TypeScript

TypeScript также используется в веб-разработке, но предпочитается для крупных проектов, где важна поддержка и читаемость кода. Примеры использования TypeScript:

  • Angular: популярный фреймворк для создания одностраничных приложений.
  • Slack: приложение для командного общения, использующее TypeScript для улучшения структуры кода.
